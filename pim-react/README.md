# PIM Studio — React App

Converted from HTML mockups. Uses Vite + React 18 + React Router v6 + Tailwind CSS v3.

## Dev
```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Pages
- / — Dashboard / Navigation Hub
- /products — Product catalogue
- /product-sets — Product sets
- /materials — Materials
- /image-engine — Image Engine
- /document-hub — Document Hub
- /social-campaign — Social Campaign
- /publish-flow — Publish Flow
- /settings — Settings
